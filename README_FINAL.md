# Sistema de Gestión de Inventario (SIGI Enterprise)

## Descripción

Sistema web desarrollado como proyecto integrador para la carrera de Sistemas de Información y Ciberseguridad del Instituto Tecnológico Superior España.

## Tecnologías utilizadas

- PHP 8.2
- SQL Server
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Composer

## Funcionalidades

- Inicio de sesión
- Gestión de usuarios
- Roles y permisos
- Categorías
- Productos
- Entradas de inventario
- Salidas de inventario
- Consulta de existencias
- Dashboard
- Reportes Excel

## Arquitectura

- MVC
- Programación Orientada a Objetos
- Principios SOLID
- Prepared Statements
- Protección CSRF
- Hash Argon2id

## Instalación

1. Clonar el repositorio.
2. Ejecutar:

```bash
composer install
```

3. Configurar el archivo `.env`.

4. Restaurar la base de datos.

5. Abrir:

```
http://localhost:81/SIGI
```
